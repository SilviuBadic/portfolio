Projects

YouTube Layout Remake

This is one of my first projects. I focused on the markup language HTML. I put into practice some basic concepts, together with adding some CSS rules over it. Here I've tried to remake the YouTube interface by maintaing the same fonts, colors, text sizes, the spaces between elements. I've added a few pictures from my trips as well, in order to give it a custom look.

Other than practicing with DIVS, It was the first time working with grid display, as well as implementing the CSS @media rule and the relative and absolute positions.

Note: It was a good exercise to practice some basic concepts. The idea was to practice by being focused on the main layout, how it looks when you open the YouTube site.

Rock-Paper-Scissors

This is one of my first projects using JavaScript. It is a classic game in which I was mainly focused on the conditionals: if, else, else if.

I also focused on the Outcome of the game, while playing against the computer. My choice is in the center, while the outcome(my choice plus computer's choice) is in the left part of the screen (text plus two small thumbnails). I used HTML and CSS for the layout. I've used JavaScript for the Reset and AutoPlay buttons as well.

Note: Exiting the game or refreshing the site won't delete the result. It was a good exercise. It was a good project to practice, to implement some more complex features than before, making my game a bit more dynamic.

Traveling Around The World

I deepened my knowledge of JavaScript in this WebSite by adding more dynamism using CSS and JavaScript.

I focused on displaying photographies from my travels while using LightBox as a tool. I used JavaScript in order to display the arrows from the section About me and the animations that comes with that. It was a project from which I learned how to organize myself along with my folder and all the files I need. I've used @keyframes in order to control my animations.

Note: It is an open project in which I am still adding pictures from my travels. The main idea was to learn more JavaScript, to add more interactivity and to develop new skills.
